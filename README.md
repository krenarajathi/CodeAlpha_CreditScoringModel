
# Credit Scoring Model

## Objective
Predict whether a loan applicant is a good or bad credit risk using financial and personal data.

## Algorithm
Random Forest Classifier

## Features
- Credit Duration
- Credit History
- Purpose of Loan
- Credit Amount
- Savings
- Employment Status
- Installment Rate
- Age
- Existing Credits
- Housing & Property Details

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
